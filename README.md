# book_bot
Телеграмм бот - который помогает быстро назодить книги по названию автору и описаню, а так же выдает рекомендации по жанрам и настроению

Используемые технологии 
- Python
- pyTelegramBotAPI
- Requests
- Google Books API

Запуск 
Установать библиотеки pip install pyTelegramBotAPI requests
API token 
python3 book_bot.py

Бот получает данные о книгах через Google Books API, обрабатывает JSON-ответы и показывает пользователю название , автора, описание и ссылку. есть созранение книг в ибранное и индивидуальные рекомендации.
